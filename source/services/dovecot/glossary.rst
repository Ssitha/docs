Dovecot Glossary
################

.. glossary::
  :abbr:`MTA`
    Mail Transport Agent: handles mail server to server (e.g. other domains).

  :abbr:`MDA`
    Mail Delivery Agent: handles user access to email (e.g. IMAP).

  :abbr:`MUA`
    Mail User Agent: user client to check email (e.g. thunderbird/outlook).
